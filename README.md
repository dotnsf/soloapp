# 'Solo' app

## Overview

HashChain 'Solo' sample application

## Pre-requisite before install

- HashChain 'Solo'

    - https://github.com/dotnsf/hashchainsolo


## Install & Setup

- Install dependencies:

    - `$ npm install`

- Run

    - `$ node app`
